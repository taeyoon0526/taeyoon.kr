피싱 메일 7가지 패턴과 대처법 (일반인용)

피싱은 기술보다 심리를 노립니다. 이 글은 일반인이 실제로 가장 많이 당하는 피싱 메일 패턴 7가지를 정리하고, 즉시 써먹는 대처법만 담았습니다.

패턴 1. “계정이 정지됩니다” 긴급 경고

  • 공포/긴급 프레임
  • 대처: 메일 링크 클릭 ❌ → 공식 사이트 직접 접속

패턴 2. 결제·환불·영수증 위장

  • 실제 서비스와 매우 유사한 디자인
  • 대처: 결제 내역은 앱/공식 페이지에서만 확인

패턴 3. 택배·배송 문제

  • 짧은 링크 + “주소 확인 필요”
  • 대처: 운송장 번호 직접 조회

패턴 4. 상금·이벤트 당첨

  • 기억나지 않는 이벤트
  • 대처: 참여한 적 없으면 100% 무시

패턴 5. 회사/학교 내부 공지 사칭

  • 첨부파일(.zip/.html) 포함
  • 대처: 내부 포털에서 재확인

패턴 6. 비밀번호 재설정 요청

  • 최근 요청한 적 없음
  • 대처: 링크 클릭 ❌ → 공식 로그인 페이지 직접 방문

패턴 7. 지인 사칭(메일 해킹 연계)

  • 짧은 문장 + 링크/파일
  • 대처: 다른 채널로 진짜 여부 확인

핵심 대처 원칙 3가지

  • 링크는 누르기 전에 도메인 확인
  • 긴급/공포 프레임이면 일단 멈춤
  • 삭제 전에 “신고”
